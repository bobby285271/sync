    Announcements

    / Since Build 262, SDDM has been replaced by LightDM. As auto-login for user live is enabled, LightDM Greeters are not pre-installed. User live has been added to the autologin user group.

    / A special GNOME 3.36 build has been uploaded, it provides all packages included in the official Archiso as well as the gnome package group and the gnome-extra package group.

    / Since Build 255, the live user has been added to the sudoers list.

    / You will need a Baidu account and a BaiduNetdisk application to download the files. The ISO images are specially optimized for Chinese users, so Chinese locale settings are enabled by default. Feel free to configure it and relogin to XFCE after that if necessary.
